# Versioning

Current version: `v0.1 Edge Demo`.

## Roadmap

| Version | Target |
|---|---|
| `v0.1` | Local single-camera edge demo |
| `v0.2` | Validated thresholds from real classroom footage |
| `v0.3` | Multi-camera edge server with `camera_id` and `classroom_id` |
| `v0.4` | Cloud metadata sync |
| `v0.5` | Next.js/React cloud dashboard |
| `v1.0` | Pilot-ready school deployment |

## Git Policy

- Keep `main` as the working demo branch until a production branch exists.
- Commit source and docs only.
- Do not commit local databases, clips, photos, uploads, model weights, or eval
  videos.
- Validate before push.
